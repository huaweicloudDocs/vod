# 密钥查询<a name="vod_04_0100_0"></a>

## 功能介绍

终端播放HLS加密视频时，向租户管理系统请求密钥，租户管理系统先查询其本地有没有已缓存的密钥，没有时则调用此接口向VOD查询。该接口的具体使用场景请参见[通过HLS加密防止视频泄露](https://support.huaweicloud.com/bestpractice-vod/vod_10_0004.html)。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=VOD&api=ShowAssetCipher)中调试该接口。

## URI

GET /v1.0/\{project\_id\}/asset/ciphers
