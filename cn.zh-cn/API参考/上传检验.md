# 上传检验<a name="vod_04_0050"></a>

## 功能介绍

校验媒资文件是否已存储于视频点播服务中。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=VOD&api=CheckMd5Duplication)中调试该接口。

## URI

GET /v1.0/\{project\_id\}/asset/duplication

## 请求示例

```
GET
https://{endpoint}/v1.0/{project_id}/asset/duplication?md5={md5}&size={size}
```

## 响应示例

**状态码： 200**

处理成功返回。

```
{
  "is_duplicated" : 1,
  "asset_ids" : [ "aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" ]
}
```

**状态码： 400**

处理失败返回。

```
{
  "error_code" : "VOD.10053",
  "error_msg" : "The request parameter is illegal, illegal field: {xx}."
}
```
