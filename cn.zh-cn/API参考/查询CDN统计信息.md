# 查询CDN统计信息<a name="vod_04_0103"></a>

## 功能介绍

查询CDN的统计数据，包括流量、峰值带宽、请求总数、请求命中率、流量命中率。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=VOD&api=ShowCdnStatistics)中调试该接口。

## URI

GET /v1.0/\{project\_id\}/asset/cdn-statistics

## 请求示例

```
GET
https://{endpoint}/v1.0/{project_id}/asset/cdn-statistics?domain=www.example.com&stat_type=cdn_bw
```

## 响应示例

**状态码： 200**

处理成功返回。

```
{
  "start_time" : "string",
  "interval" : 0,
  "values" : [ 0 ]
}
```

**状态码： 400**

处理失败返回。

```
{
  "error_code" : "VOD.10053",
  "error_msg" : "The request parameter is illegal, illegal field: {xx}."
}
```
