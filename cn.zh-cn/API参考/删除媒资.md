# 删除媒资<a name="vod_04_0024"></a>

## 功能介绍

删除媒资。

## 接口约束

正在处理中的媒资不支持删除，包括转码中、审核中、截图中等。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=VOD&api=DeleteAssets)中调试该接口。

## URI

DELETE /v1.0/\{project\_id\}/asset

## 请求示例

```
DELETE  https://{endpoint}/v1.0/{project_id}/asset?asset_id={asset_id}
```

## 响应示例

**状态码： 200**

处理成功返回。

```
{
  "delete_result_array" : [ {
    "asset_id" : "f488337c31c8e4622f1590735b134c65",
    "status" : "DELETED"
  } ]
}
```

**状态码： 403**

处理失败返回。

```
{
  "error_code" : "VOD.10062",
  "error_msg" : "Media asset or resource does not exist, please check."
}
```
