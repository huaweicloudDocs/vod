# 查询媒资列表<a name="vod_04_0203"></a>

## 功能介绍

查询媒资列表，列表中的每一条记录包含媒资的概要信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=VOD&api=ListAssetList)中调试该接口。

## URI
