# 查询托管任务<a name="ListTakeOverTask"></a>

## 功能介绍

查询OBS存量托管任务列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=VOD&api=ListTakeOverTask)中调试该接口。

## URI

GET /v1.0/\{project\_id\}/asset/obs/host/stock/task
