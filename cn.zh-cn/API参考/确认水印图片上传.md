# 确认水印图片上传<a name="ConfirmImageUpload"></a>

## 功能介绍

确认水印图片上传状态。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=VOD&api=ConfirmImageUpload)中调试该接口。

## URI

POST /v1.0/\{project\_id\}/watermark/status/uploaded
