# 视频拼接<a name="ZH-CN_TOPIC_0137148888"></a>

## 功能描述<a name="zh-cn_topic_0128109935_zh-cn_topic_0127939728_section114814192538"></a>

将多个视频（如片头、正片与片尾）拼接成新视频文件，并将新视频文件添加到点播系统中。

## 约束限制<a name="section1996645413524"></a>

-   目前仅支持MP4、FLV格式的视频任一种格式拼接或者混合拼接。
-   一次拼接任务最多支持同时拼接10个视频。

## 请求URI<a name="zh-cn_topic_0128109935_zh-cn_topic_0127939728_section5241024145313"></a>

POST /v1.0/\{[project\_id](获取项目ID.md)\}/asset/concat
