# 媒资发布取消<a name="vod_04_0021"></a>

## 功能介绍

将媒资设置为未发布状态。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=VOD&api=UnpublishAssets)中调试该接口。

## URI

POST /v1.0/\{project\_id\}/asset/status/unpublish
