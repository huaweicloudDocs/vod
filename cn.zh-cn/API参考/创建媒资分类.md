# 创建媒资分类<a name="vod_04_0028"></a>

## 功能介绍

创建媒资分类。

## 接口约束

最大支持三级分类，每个分类最多支持创建128个子分类。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=VOD&api=CreateAssetCategory)中调试该接口。

## URI

POST /v1.0/\{project\_id\}/asset/category

## 请求示例

```
POST https://{endpoint}/v1.0/{project_id}/asset/category

{
  "name" : "电影"
}
```

## 响应示例

**状态码： 200**

处理成功返回。

```
{
  "id" : 334,
  "name" : "电影",
  "parent_id" : 0,
  "level" : 1,
  "projectId" : "58411d8df9064*****75d75b54e01358"
}
```

**状态码： 400**

处理失败返回。

```
{
  "error_code" : "VOD.10053",
  "error_msg" : "The request parameter is illegal, illegal field: {xx}."
}
```
